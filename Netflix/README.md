# Netflix Recommendation System

## Objective

Build a movie recommendation system that suggests movies to users based on their preferences and historical ratings.

## Dataset Information

The dataset contains:

* User IDs
* Movie IDs
* Movie Names
* Genres
* User Ratings

## Business Questions Solved

1. Identify the most popular and highly rated movie genres.
2. Find which genres have received the highest and lowest ratings.
3. Develop a recommendation system that suggests the most suitable movies for users.
4. Analyze user preferences and movie rating patterns.

## Techniques Used

* Exploratory Data Analysis (EDA)
* Collaborative Filtering
* Singular Value Decomposition (SVD)
* Recommendation Engine

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Surprise Library
* Scikit-learn

## Outcome

Developed a personalized movie recommendation system capable of recommending movies based on user rating behavior and preferences.
