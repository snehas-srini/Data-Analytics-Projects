# Pizza Sales Analysis Dashboard

## Objective

Analyze pizza sales data to identify revenue trends, best-selling products, customer ordering patterns, and key business insights using SQL and Power BI.

## Techniques Used

* KPI Analysis
* Sales Trend Analysis
* Revenue Analysis
* Product Performance Analysis
* Customer Order Pattern Analysis
* Dashboard Development

## Tools & Technologies

* SQL
* Power BI

## Outcome

Developed an interactive Power BI dashboard to analyze pizza sales performance, identify top-selling products, track key business metrics, and provide actionable business insights.
