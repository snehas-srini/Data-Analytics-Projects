# Data-Analytics-Projects

A collection of Data Analytics and Data Science projects demonstrating skills in SQL, Python, Pandas, Machine Learning, Recommendation Systems, Time Series Forecasting, and Power BI.

## Projects

### 1. Walmart Sales Forecasting

**Objective:** Analyze Walmart sales data and forecast sales for the next 12 weeks.

**Techniques Used:**

* Exploratory Data Analysis (EDA)
* Outlier Analysis
* Correlation Analysis
* Sales Forecasting

**Tools & Libraries:**

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

### 2. Netflix Recommendation System

**Objective:** Build a movie recommendation system using user ratings and movie data.

**Techniques Used:**

* Collaborative Filtering
* Singular Value Decomposition (SVD)
* Recommendation Engine

**Tools & Libraries:**

* Python
* Pandas
* NumPy
* Surprise Library
* Scikit-learn

---

### 3. COVID-19 Time Series Forecasting

**Objective:** Analyze COVID-19 trends and forecast future cases.

**Techniques Used:**

* Time Series Analysis
* Trend Analysis
* Forecasting using Prophet

**Tools & Libraries:**

* Python
* Pandas
* Matplotlib
* Prophet

---

### 4. Pizza Sales Analysis Dashboard

**Objective:** Analyze pizza sales data to identify revenue trends, best-selling products, and key business insights using SQL and Power BI.

**Techniques Used:**
- Sales Analysis
- Revenue Analysis
- KPI Tracking
- Product Performance Analysis
- Customer Order Trends

**Tools & Technologies:**
- SQL
- Power BI
