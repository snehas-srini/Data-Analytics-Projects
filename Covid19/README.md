# COVID-19 Time Series Forecasting

## Objective

Analyze COVID-19 trends and forecast future cases using time series forecasting techniques.

## Techniques Used

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Time Series Analysis
* Trend Analysis
* Forecasting using Prophet

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Plotly
* Prophet

## Outcome

Built a time series forecasting model using Prophet to analyze COVID-19 trends and predict future case counts based on historical data patterns.
